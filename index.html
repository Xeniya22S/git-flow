Git-flow

  • git status- проверяем какие файлы мы можем сохранить (измененные файлы)
  • git add . - фиксируем список для добавления в git (. -это добавление всех измененных файлов)
  • git commit -m "название commit" - создаем название нашему комментарию
  • git push origin main - отправляем измененные файлы на github
  • закрепляем материал
